# python-regex-samples
# Python 正規表現サンプル

このリポジトリは、Pythonの正規表現に関する発表で使用したサンプルコードをまとめたものです。

## 動作環境

- Python 3.x

## サンプル一覧

| ファイル | 内容 |
|----------|------|
| 01_phone.py | 電話番号の検索 |
| 02_group.py | グループ化 |
| 03_escape.py | 脱出文字 |
| 04_pipe.py | `|`（または）の利用 |
| 05_findall.py | `findall()` |
| 06_sub.py | `sub()` |
| practice.py | 練習問題 |

## 実行方法

```bash
python 01_phone.py
```

## 参考

Python標準ライブラリ `re` モジュールを使用しています。
